Getting started

Dokumentasi statis ini menampilkan endpoint inti, contoh, dan pola integrasi yang direkomendasikan untuk Capella ERP API v6. Semua endpoint biasanya memerlukan header Authorization: Bearer <token>.

Base URL

https://api.capellaerp.id/api/v6

Content types

Gunakan application/json untuk request dan response.

Authentication

JWT Bearer token. Tukarkan kredensial untuk mendapatkan access token.

POST /api/v6/auth/token
Dapatkan access token (username/password atau API key).
v6
Example 
curl -X POST https://api.capellaerp.id/api/v6/auth/token \
  -H "Content-Type: application/json" \
  -d '{"username":"YOUR_USER","password":"YOUR_PASS"}'

Sales

Endpoints untuk pelanggan, invoice, dan sales orders.

GET /api/v6/sales/orders
List sales orders dengan pagination dan filter (status, date_from, date_to).
v6
Example 
curl -H "Authorization: Bearer YOUR_TOKEN" \
  'https://api.capellaerp.id/api/v6/sales/orders?page=1&pageSize=25'
POST /api/v6/sales/orders
Buat sales order baru. Body JSON berisi items dan referensi customer.
v6
Example 
curl -X POST https://api.capellaerp.id/api/v6/sales/orders \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"customerId":"CUST123","items":[{"sku":"PROD-1","qty":2}]}'

Inventory

Manajemen stok: items, warehouses, stock adjustments.

GET /api/v6/inventory/items
Search & list inventory items. Support q, category, warehouseId.
v6
Example 
curl -H "Authorization: Bearer YOUR_TOKEN" \
  'https://api.capellaerp.id/api/v6/inventory/items?q=batang'
POST /api/v6/inventory/adjustments
Buat stock adjustment (positive/negative).
v6
Example 
curl -X POST https://api.capellaerp.id/api/v6/inventory/adjustments \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"warehouseId":"WH01","lines":[{"sku":"PROD-1","qty":-3}]}'

Finance

General ledger, journals, dan reporting endpoints.

GET /api/v6/finance/journals
Ambil journal entries dengan filter tanggal.
v6
Example 
curl -H "Authorization: Bearer YOUR_TOKEN" \
  'https://api.capellaerp.id/api/v6/finance/journals?date_from=2025-01-01&date_to=2025-01-31'

Examples & SDK hints

Potongan kode cepat untuk tugas umum.

Login (JS)

async function login(user, pass){
  const res = await fetch('https://api.capellaerp.id/api/v6/auth/token',{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify({username:user,password:pass})})
  return res.json()
}

Create Sales Order (JS)

async function createOrder(token, payload){
  const res = await fetch('https://api.capellaerp.id/api/v6/sales/orders',{method:'POST',headers:{'Content-Type':'application/json','Authorization':'Bearer '+token},body:JSON.stringify(payload)})
  return res.json()
}
SDK advice: buat wrapper kecil untuk auth, pagination, retry. Gunakan TypeScript untuk konsumen tipe aman.
